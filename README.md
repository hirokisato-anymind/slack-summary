# 1on1マネジメント支援システム

Claude Codeを使って、メンバーとの1on1準備・成長追跡・フィードバックを効率化するシステムです。

## クイックスタート

### 1. メンバーファイルを作成する

```
context/members/_template.md をコピーして
context/members/{名前}.md を作成する
```

プロフィール・目標・評価軸を記入する（最初は箇条書きでOK）。

### 2. 1on1前に準備資料を生成する

```
/1on1-prep {名前}
```

Claude Codeが自動で以下を実施する:
- Slackの直近2週間の発言を収集・分析
- 前回アクションアイテムの進捗を確認
- 褒めるべき具体的行動を抽出
- 成長の変化を時系列で整理
- フェーズ進捗を確認
- 1on1アジェンダ案を生成

### 3. 1on1後に気づきを追記する

1on1で感じたこと（声のトーン、モチベーション、定性的な観察）をメンバーファイルに追記する。
これが次回の分析精度を高める。

---

## 連携ツール

| ツール | 用途 | 必須/任意 |
|--------|------|---------|
| Slack MCP | 発言・行動の収集 | 推奨 |
| Circleback / AI議事録ツール | MTG議事録のテキスト化 | 推奨 |
| Notion | メンバーへの目標共有 | 任意 |
| Google Calendar | 今後の予定確認 | 任意 |

---

## ファイル構成

```
1on1-management/
├── CLAUDE.md                      # Claude Codeへの基本指示
├── README.md                      # このファイル
├── context/
│   └── members/
│       ├── _template.md           # メンバーファイルのテンプレート
│       └── {名前}.md              # 各メンバーの成長追跡ファイル
└── .claude/
    └── commands/
        └── 1on1-prep.md           # /1on1-prep コマンド定義
```

---

## 活用のコツ

- **ファイルを育てる**: 1on1のたびに追記することで精度が上がる
- **気づいたらすぐ記録**: 本人に伝える前にまずClaude Codeのファイルに書く
- **AIの出力に温度を乗せる**: 事実はAIが拾い、伝え方は自分の言葉で
- **短期の波に惑わされない**: 時系列の変化を信じる
